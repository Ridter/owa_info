
# owa_info
用于对Exchange的基础信息进行搜集的小脚本。

## Usage
```
usage: owa_info.py [-h] -u URL [-t TIMEOUT] [-d]

OWA Info Scanner

optional arguments:
  -h, --help            show this help message and exit
  -u URL, --url URL     Exchange OWA URL
  -t TIMEOUT, --timeout TIMEOUT
                        Timeout
  -d, --debug           Print Debug info
```
